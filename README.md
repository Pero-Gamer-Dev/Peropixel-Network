## 🌐 Peropixel Network  

Peropixel Network is a custom-built Minecraft server network focused on delivering optimized performance and unique gameplay systems.

---

### 🚀 Overview  
Peropixel is designed with scalability and custom development in mind, combining advanced plugin systems, optimized configurations, and engaging gamemodes.

---

### 🎮 Gamemodes  
- Skyblock  
- Lifesteal  
- Bedwars  
- Skywars
